

## 7. wger

### Repository
> [wger Repo](https://github.com/wger-project/wger/issues)

### Homepage
> [wger Homepage](https://wger.de/)

### Product
> Fitness tracker that allows users to create workout routines, plan meals, and track progress. Offers an API for other apps and the option to set up a private instance to run your own server.

### Target User Community
> Users interested in physical fitness and health.

### Activity
> 9AM this morning, 8576 commits.

### Getting Started
> They have a [contribution guide](https://wger.readthedocs.io/en/latest/contributing.html). It recommends checking the roadmap for the next release. There are three main repos: backend, frontend, and mobile app; all with links to setting up developer environments and the release process.

### Welcoming to Newcomers
> The guides feel very beginner-friendly. They use the "good first issue" label, and the conversation feels very welcoming and friendly.
